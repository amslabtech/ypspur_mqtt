# ypspur_mqtt
MQTT wrapper for YP-Spur

## Dependencies
- [yp-spur](https://github.com/openspur/yp-spur)
  - one of the following is required:
    - build and install yp-spur
    - clone yp-spur to the root directory of this project


## Build
```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install
```
